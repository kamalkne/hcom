# hcom
App build in Angular 4, TypeScript, webpack 3 with scroll spy feature.

# Setup

1. clone Repository
2. Open cmd
3. cd hcom
4. npm install
5. npm start - This process will create dist folder
6. cd server
7. node server.js
8. Open http://localhost:3001 in your browser
